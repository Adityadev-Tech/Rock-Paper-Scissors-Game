# Rock-Paper-Scissors-Game
This is simple rock paper scissors game build using concepts of WebDevelopment
<p>Rock Paper Scissors is a simple and interactive game built using HTML, CSS, and JavaScript. It lets users play the classic Rock Paper Scissors game against the computer with a clean interface and easy-to-use controls. This project was created to practice core front-end development skills, including DOM manipulation, event handling, and basic game logic</p>
